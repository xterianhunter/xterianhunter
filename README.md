## Hi there 👋

# 💫 About Me:
I’m a Computer Science undergraduate passionate about AI, Machine Learning, and Intelligent Systems, with hands-on experience in real-world problem solving. My work spans Computer Vision (YOLO-based detection systems), Traffic Forecasting using Graph Neural Networks (ASTGCN, STGCN, DCRNN), and Cyber Attack Detection using LSTM & Random Forest on CICIDS2017.

I enjoy building end-to-end ML pipelines — from data preprocessing and model training to deployment and visualization — and I actively experiment with tools like PyTorch, TensorFlow, SUMO, and OpenCV. I’m particularly interested in smart transportation systems, geospatial AI, and scalable ML architectures.

Currently, I’m focused on advancing deep learning research, optimizing model performance, and developing impactful AI-driven solutions that bridge theory with real-world applications.
